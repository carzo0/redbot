RedBot - Términos de Servicio

1. Introducción

Al invitar, acceder o usar RedBot, aceptas estos Términos de Servicio, así como los Términos de Servicio de Discord y las leyes aplicables.

Si no estás de acuerdo con alguna parte, no debes usar RedBot.

RedBot puede actualizar estos Términos en cualquier momento. El uso continuado implica aceptación de los cambios.

2. Descripción del Servicio

RedBot es un bot público de Discord desarrollado con JavaScript y el framework Erine.

Proporciona funciones de utilidad y entretenimiento, incluyendo:

  • Mostrar información pública de usuarios (ID, roles, fecha de creación)
  • Mostrar avatares, banners y logos de servidores
  • Comandos de ayuda, ping e información

No realiza moderación automática, transacciones financieras ni control administrativo fuera de los permisos otorgados por los administradores del servidor.

3. Elegibilidad

  • Debes cumplir con los TOS y Guidelines de Discord.
  • Debes tener al menos 13 años o la edad mínima requerida en tu país.
  • Debes contar con autorización del dueño o administrador del servidor para invitar y usar el bot.

RedBot puede restringir o eliminar el acceso de usuarios o servidores que violen estas condiciones.

4. Uso Aceptable

No debes:

  • Usar el bot para acoso, spam o actividades ilegales
  • Intentar explotar, sobrecargar, revertir ingeniería o interrumpir su funcionamiento
  • Copiar, modificar o redistribuir RedBot sin permiso
  • Recopilar o exponer información privada de usuarios
  • Violar las políticas de Discord, API o leyes aplicables

El incumplimiento puede resultar en restricción permanente del uso de RedBot.

5. Datos y Procesamiento AI

  • RedBot procesa únicamente los datos necesarios para ejecutar comandos mediante la API oficial de Discord.
  • El contenido de mensajes solo se procesa temporalmente cuando:
    • Un usuario invoca un comando con prefijo o slash
    • Un usuario responde a un mensaje generado por un comando AI
  • El contexto de conversación AI se guarda temporalmente en memoria y se descarta automáticamente tras expiración o reinicio del bot.
  • RedBot no almacena permanentemente mensajes, attachments o embeds.
  • Los datos de configuración (como prefijos) pueden guardarse temporalmente.
  • No se venden, comparten ni monetizan datos de usuarios.

  • El desarrollador no es responsable de pérdida de datos, interrupciones o mal uso por usuarios
