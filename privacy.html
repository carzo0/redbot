🔐 Política de Privacidad - RedBot

Al invitar o usar RedBot, aceptas automáticamente esta Política de Privacidad y las condiciones descritas a continuación.

Información Recopilada y Procesada

1. Registro del Servidor

Cuando RedBot se agrega a un servidor, puede almacenar el ID del servidor para propósitos de configuración (por ejemplo, ajustes de prefijo personalizados).

2. Uso de Comandos

Cuando un usuario ejecuta un comando, el bot procesa:

  • ID del usuario
  • ID del servidor
  • ID del canal
  • Nombre del comando
  • Parámetros del comando

Esta información se usa únicamente para ejecutar la función solicitada.

3. Procesamiento de Contenido de Mensajes

  • Por defecto, RedBot NO lee ni almacena todos los mensajes del servidor.
  • El contenido de los mensajes se procesa solo cuando:
    • Un usuario invoca explícitamente un comando con prefijo o slash.
    • Un usuario responde directamente a un mensaje generado por un comando AI.
  • Al usar el comando AI ("ask"), el contenido se procesa temporalmente para generar la respuesta.
  • El contexto de conversación AI se guarda temporalmente en memoria y se descarta automáticamente tras la expiración o reinicio del bot.
  • RedBot NO almacena permanentemente contenido de mensajes, archivos adjuntos o embeds.

Procesamiento de AI

  • Las respuestas de AI pueden ser generadas usando proveedores de AI externos.
  • Solo se transmite el contenido enviado explícitamente a través del comando AI.
  • RedBot NO usa el contenido de los mensajes para entrenar modelos AI propios.
  • El contenido del usuario no se vende, comparte ni monetiza.

Almacenamiento de Datos

  • RedBot no mantiene bases de datos permanentes de contenido de mensajes.
  • Datos de configuración (como prefijo del servidor) pueden almacenarse temporalmente.
  • Los datos en memoria se eliminan automáticamente tras la expiración de la sesión.
  • No se mantienen registros a largo plazo de los mensajes de los usuarios.

Compartición de Datos

RedBot no vende, alquila ni distribuye datos de usuarios. Los datos pueden compartirse únicamente con:

  • API oficial de Discord (requerida para funcionamiento)
  • Proveedores de servicios AI (solo cuando se usa el comando AI)
  • Autoridades legales si es requerido por ley

Solicitud de Eliminación de Datos

Como RedBot no almacena permanentemente contenido personal de mensajes, la mayoría de los datos se eliminan automáticamente.

Los administradores del servidor pueden eliminar datos de configuración removiendo el bot del servidor.

Para solicitudes de eliminación permanente, contacta al desarrollador a través del servidor de soporte:

Servidor de soporte de RedBot

Requisito de Edad

RedBot no está destinado a usuarios menores de 13 años, de acuerdo con los Términos de Servicio de Discord.

Cambios en la Política

Esta Política de Privacidad puede actualizarse en cualquier momento. El uso continuado de RedBot implica aceptación de la política actualizada.
