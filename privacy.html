Política de Privacidad

Al invitar o usar RedBot, aceptás esta política. Acá te explicamos exactamente qué hacemos con tus datos, sin letra chica.

01

Información recopilada

Registro del servidor

Cuando RedBot se agrega a un servidor, puede almacenar el ID del servidor para propósitos de configuración, como el prefijo personalizado.

Uso de comandos

Cuando ejecutás un comando, el bot procesa temporalmente:

  • ID del usuario
  • ID del servidor y del canal
  • Nombre del comando y sus parámetros

Esta información se usa únicamente para ejecutar la función solicitada y no se almacena.

Contenido de mensajes

RedBot NO lee ni almacena todos los mensajes del servidor. El contenido solo se procesa cuando vos explícitamente usás un comando.

El contenido se procesa únicamente cuando:

  • Invocás un comando con prefijo o slash
  • Respondés directamente a un mensaje generado por el comando AI
02

Procesamiento de IA

Al usar el comando /ask, el contenido de tu mensaje se transmite temporalmente a un proveedor de IA externo para generar la respuesta.

  • El contexto de conversación se guarda en memoria y se descarta automáticamente tras 10 minutos de inactividad o al reiniciar el bot
  • RedBot NO usa tu contenido para entrenar modelos propios
  • Tu contenido no se vende, comparte ni monetiza
03

Almacenamiento de datos

  • No hay bases de datos permanentes de contenido de mensajes
  • Los datos de configuración (como prefijo del servidor) se almacenan en base de datos
  • Los datos en memoria se eliminan automáticamente
  • No hay registros a largo plazo de mensajes de usuarios
04

Compartición de datos

RedBot no vende, alquila ni distribuye datos. Los datos pueden compartirse únicamente con:

  • API oficial de Discord (requerida para el funcionamiento)
  • Proveedores de IA (solo cuando usás el comando /ask)
  • Autoridades legales si es requerido por ley
05

Eliminación de datos

Como RedBot no almacena permanentemente contenido de mensajes, la mayoría de datos se eliminan automáticamente.

Los administradores pueden eliminar datos de configuración removiendo el bot del servidor. Para solicitudes específicas, contactá al desarrollador:

Servidor de soporte de RedBot →

06

Requisitos y cambios

RedBot no está destinado a usuarios menores de 13 años, de acuerdo con los Términos de Servicio de Discord.

Esta política puede actualizarse en cualquier momento. El uso continuado de RedBot implica aceptación de la versión actualizada.
